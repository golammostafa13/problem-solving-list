## Approach-1
```py
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        for i in range(len(nums)):
            for j in range(i+1, len(nums)):
                firstNumber = nums[i]
                secondNumber = target - firstNumber
                if nums[j] == secondNumber:
                    return [i, j]
```
